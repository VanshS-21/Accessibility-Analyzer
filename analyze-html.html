🔍 Understanding Results

  • • 🔴 Critical: Blocks users completely
  • • 🟠 Serious: Major accessibility barriers
  • • 🟡 Moderate: Causes difficulties
  • • 🟢 Minor: Enhancement opportunities

💡 Pro Tip: Each issue includes WCAG success criteria, real-world impact examples, and step-by-step fixes to help you learn web accessibility!

📝 HTML Analysis Tips

  • • Include the complete HTML structure including DOCTYPE, html, head, and body tags
  • • Make sure to include any relevant CSS that affects accessibility (like hiding content)
  • • The analyzer works best with complete, well-formed HTML documents
  • • Check both the structure and semantic markup of your HTML

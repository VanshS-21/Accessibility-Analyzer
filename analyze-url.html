Alternative: Manual Analysis

If the URL analysis doesn't work due to CORS restrictions, you can:

  1. Visit the website you want to analyze
  2. Right-click and select "View Page Source" or press Ctrl+U (Cmd+U on Mac)
  3. Copy the HTML source code
  4. Use our HTML Analysis feature to paste and analyze the code

Tips for URL Analysis

  • • Make sure to include the protocol (http:// or https://)
  • • Some websites may block cross-origin requests for security reasons
  • • Public websites and demo sites typically work better than private or secured sites
  • • If analysis fails, try the HTML analysis method described above
